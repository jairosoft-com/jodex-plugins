# Prompts

Reusable markdown prompt fragments shared across commands and skills.

## Format

Each `.md` file contains a prompt template that can be referenced by commands or skills.
No special frontmatter required — just markdown content.

## When to use

- Shared wiki page creation guidelines across ingest and query skills
- Common frontmatter validation rules
- Reusable cross-referencing instructions

## Reference

See jx-qa plugin's `prompts/ABOUT.md` for format details.
