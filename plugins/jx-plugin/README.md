# jx-plugin

Plugin skeleton — scaffolding, validation, and publishing tools (in development)

## Status

This is a plugin skeleton. It contains package metadata, placeholder component directories, and a marketplace entry only.

## Next Step

Add the first real skill with:

```bash
/jx-skill:create --plugin jx-plugin
```

## Local Development

```bash
claude --plugin-dir /path/to/jodex-plugins/plugins/jx-plugin
claude plugin marketplace add /path/to/jodex-plugins --scope project
claude plugin install jx-plugin@jodex-plugins
```
