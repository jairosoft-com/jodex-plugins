# Prompts

Reusable markdown prompt fragments shared across commands and skills.

## Format

Each `.md` file contains a prompt template that can be referenced by commands or skills.
No special frontmatter required — just markdown content.

## When to use

- Common instructions referenced by multiple commands
- Review templates with consistent structure
- Shared test-writing guidelines for both extract and generate

## Reference

See Codex plugin's `prompts/adversarial-review.md` and `prompts/stop-review-gate.md` for working examples.
