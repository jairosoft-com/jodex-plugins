# Codex Desktop

Desktop application for Codex. Shares the [[Plugin Architecture]] and plugin registry (`~/.claude/plugins/installed_plugins.json`) with [[Claude Code CLI]] and [[Claude Code Desktop]].

## Plugin Compatibility

Plugins installed via [[Claude Code CLI]] or added to the registry are available in Codex Desktop. Same [[Marketplace]] system applies.

See [[Plugin Architecture#Compatible Platforms]] for full list.
