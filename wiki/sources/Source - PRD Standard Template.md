## Summary

Template for the standard mode PRD. Extends the lite template with additional sections for design considerations, technical considerations, and more detailed non-functional requirements (including scalability). Adds E2E test AC for UI stories and provides a global AC counter that continues across user stories.

## Key Concepts
- Standard mode: mid-weight PRD format between lite and unified
- Global AC counter continues across user stories (not per-story)
- E2E tests AC added for UI stories only
- Design considerations section (UI/UX, component reuse, interaction patterns)
- Technical considerations section (constraints, integration points, data model)
- Success metrics include measurement method
