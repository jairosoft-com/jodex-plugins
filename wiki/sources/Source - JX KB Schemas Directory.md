## Summary

Describes the schemas directory for the llm-wiki plugin. Contains JSON Schema files (draft-07 or later) that define structured output contracts for commands and skills. Use cases include validating wiki page frontmatter, typing lint report output, and defining ingest plan table schemas. Clarifies that the wiki's own configuration document (`_schema.md`) lives in the user's wiki directory, not in this plugin directory.

## Key Concepts
- JSON Schema draft-07 for structured output validation
- Wiki page frontmatter validation
- Lint report output typing
- Ingest plan table schema definition
- Separation of plugin schemas from wiki `_schema.md`
