## Summary

Reference library of Mermaid diagram patterns for use in TECH_SPEC generation. Provides ready-to-use templates for seven diagram types: C4 Context, C4 Container, Sequence, Entity-Relationship (ERD), State Machine, Flowchart (decision logic), and Deployment/Infrastructure. Includes usage guidance on when to apply each type.

## Key Concepts
- C4 Context diagram: system boundaries, actors, external dependencies
- C4 Container diagram: internal components and relationships
- Sequence diagram: component interactions over time
- ERD: data model relationships with cardinality
- State Machine diagram: entity lifecycle transitions
- Flowchart: branching logic and decision points
- Deployment diagram: infrastructure topology
- One concept per diagram principle
