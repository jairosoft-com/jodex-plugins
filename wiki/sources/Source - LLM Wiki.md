## Summary

Abstract pattern document describing how to build personal knowledge bases using LLMs. Instead of RAG-style retrieval from raw documents, the LLM incrementally builds and maintains a persistent wiki — a structured, interlinked collection of markdown files.

- Navigation via [[Index]] and [[Log]]
- [[Obsidian]] as the wiki viewer/IDE
- Connection to [[Vannevar Bush]]'s Memex (1945)
