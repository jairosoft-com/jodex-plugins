## Summary

The root README for the jodex-plugins marketplace by Jairosoft. Documents the five-plugin marketplace: [[QA Testing Plugin|jx-qa]], [[Knowledge Base Plugin|jx-kb]], [[Product Management Skills Plugin|jx-pm]], [[Developer Skills Plugin|jx-dev]], and [[Core Shared Conventions Plugin|jx-core]]. The jx-qa command surface includes `/jx-qa:extract`, `/jx-qa:generate`, `/jx-qa:browser`, and `/jx-qa:test`.

## Key Concepts
- Five-plugin Claude Code marketplace
- Four-command jx-qa surface: extract, generate, browser, test
- jx-pm depends on jx-core and jx-dev; jx-dev depends on jx-core
- Installation through `/plugin marketplace add` and `/plugin install`
- Local development with `--plugin-dir` or a local marketplace registration
- Dependency stack: Python 3, openpyxl, Node.js 18+, Playwright, Azure DevOps MCP for jx-pm:ado
