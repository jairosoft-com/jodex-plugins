## Summary

The root README for the jodex-qa-ai project — a Claude Code plugin marketplace for qa-ai that turns BRD/PRD documents into Playwright test suites. Documents the three main commands (extract, generate, browser), system requirements (Python 3 + openpyxl, Node.js 18+, Playwright, playwright-cli), and installation instructions for both Claude Code CLI and Desktop. Also covers local development setup and uninstall procedures.

## Key Concepts
- Plugin marketplace concept (`/plugin marketplace add`)
- Three-command qa-ai surface: extract, generate, browser
- Dual installation paths: CLI and Desktop
- Local development with `--plugin-dir` flag
- Dependency stack: Python 3, openpyxl, Node.js 18+, Playwright
- Plugin lifecycle: install, reload, uninstall
