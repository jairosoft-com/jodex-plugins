# Source - Spec-First Feedback

Agent feedback memory capturing a generalized lesson: read the SKILL.md before executing any jx-* skill.

## Origin

Session working on FEAT-006 (meet-email skill for [[Product Management Skills Plugin|jx-pm]]). Two back-to-back spec violations:
1. `/jx-pm:ado` — improvised ADO work item format
2. `/jx-kb:ingest` — improvised wiki ingest phases

## Key Claim

Memorizing a specific fix ("read ado.md before ADO sync") fails to prevent the same class of mistake on the next skill. The principle must be generalized: "read the SKILL.md before running any skill."
