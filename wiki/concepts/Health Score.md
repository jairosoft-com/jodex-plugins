# Health Score

A 0-100 score generated by [[Lint]] that quantifies wiki structural and content quality.

## Calculation

- Start at 100
- Each **error**: -10 (broken links, missing index entries, missing frontmatter)
- Each **warning**: -5 (orphans, unresolved conflicts, thin pages, stale ideas)
- Each **info**: -1 (stale pages, missing wikilinks, duplicate candidates)
- Floor at 0

## Finding Categories

| Severity | Examples |
|----------|----------|
| Error | Broken wikilink, page missing from index, missing frontmatter fields |
| Warning | Orphan page (no inbound links), unresolved `[!conflict]`, thin page (<3 sentences), idea raw >14 days |
| Info | Page not updated in 90+ days, entity name mentioned without wikilink, possible duplicates |

## Interpretation

A low score is only useful if the scanned page set matches the maintained wiki. If [[Raw Sources]] are counted as normal pages, provenance noise can dominate the score and hide the real maintenance signal. See [[Wiki Signal Quality]].

## Auto-Fixes

Lint offers to fix automatically (with user confirmation):
- Add missing pages to `_index.md`
- Add wikilinks to unlinked mentions
- Remove stale index entries for deleted pages
