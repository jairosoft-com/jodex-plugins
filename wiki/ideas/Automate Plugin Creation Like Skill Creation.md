# Automate Plugin Creation Like Skill Creation

Provide a scaffolding command (e.g. `/plugin:init`) that automates creation of a new plugin directory, mirroring how skill creation is already automated. The generator would produce the canonical directory layout, manifest, empty skill stubs, and boilerplate files so a new plugin is ready to develop without manual setup.

## Key Points

- Mirror the existing skill-creation automation pattern
- Generate plugin manifest, directory structure, and starter files in one step
- Reduce friction for contributors adding new plugins to the jx ecosystem
