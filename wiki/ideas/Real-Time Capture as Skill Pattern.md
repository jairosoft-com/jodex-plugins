# Real-Time Capture as Skill Pattern

meet-notes introduces a pattern not yet seen in existing jx-pm skills: real-time interactive capture during an ongoing activity. Existing skills are either batch generators (prd, spec) or data-pull snapshots (meet-pre). A capture skill needs different phase design — continuous input, incremental append, and a finalize/export step rather than the usual question → generate → write flow.
