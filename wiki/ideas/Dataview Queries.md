# Dataview Queries

Dataview is an [[Obsidian]] plugin that runs queries over page YAML frontmatter. If wiki pages have consistent frontmatter (tags, dates, source counts), Dataview can generate dynamic tables and lists.

## Potential Uses

- List all pages by source_count (which pages are most well-sourced?)
- Filter by type or tag
- Show recently updated pages
- Track idea lifecycle (raw → promoted/archived)
