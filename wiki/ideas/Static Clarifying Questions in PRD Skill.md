# Static Clarifying Questions in PRD Skill

Phase 3 clarifying questions in the PRD skill are hardcoded per mode. A healthcare PRD and an e-commerce PRD get identical questions. No way to customize or extend the question set per project or domain.

Domain-specific concerns (HIPAA, PCI, SOC2) only surface if the user picks "D. Regulatory compliance" in Q1. Consider conditional follow-up questions based on Q1 answer, or a project-level question config that adds domain-specific questions.
