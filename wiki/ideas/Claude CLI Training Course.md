# Claude CLI Training Course

Short training course covering how to use Claude Code CLI — launching from Windows and Mac terminals, navigating the help system, and core workflows.

## Motivation

- Team members need practical guidance on getting started with Claude CLI
- Different setup steps for Windows vs macOS terminals
- The built-in help system (`claude --help`, `/help`) is powerful but not obvious to new users
- A structured short course lowers the barrier to adoption

## Possible Topics

- Installing Claude CLI (npm, homebrew, manual)
- Launching Claude from Windows Terminal / PowerShell
- Launching Claude from macOS Terminal / iTerm
- Using the help page (`claude --help`, `claude -h`)
- In-session slash commands (`/help`, `/clear`, `/config`, etc.)
- Basic workflows: ask questions, edit files, run commands
- Permission modes and what they mean
- CLAUDE.md and project context
- Setting up the Anthropic API key (obtaining, configuring, environment variables) — ref: https://support.claude.com/en/articles/9267247-get-started-with-the-team-plan
- Common first-session pitfalls
